# chatbot-using-camel-rabbitmq-springboot

This is a simple chatbot using Apache Camel RabbitMQ Component + Program AB + Spring boot.

To Run the application, use the following steps:-
  1. Clone or Download the ZIP 
  2. Run the camel-rabbitmq-server application
  3. Run the camel-rabbitmq-client application
  
# Note:
  I assume that you have installed RabbitMQ in your system already and its running on port 15672. If not so, please install the RabbitMQ first and then start your application. Use this link to install RabbitMQ:- https://www.rabbitmq.com/download.html
